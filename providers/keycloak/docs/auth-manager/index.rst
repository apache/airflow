=====================
Keycloak auth manager
=====================

.. warning::
  The Keycloak auth manager is alpha/experimental at the moment and may be subject to change without warning.

The Keycloak auth manager is an authentication and authorization manager that uses Keycloak as the identity provider.
It enables you to manage users, roles, groups, and permissions entirely within Keycloak.
