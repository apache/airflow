``airflow.api.client`` is replaced by the ``airflowctl`` client

The legacy ``airflow.api.client`` module is removed. The ``airflow`` CLI now reaches the API
server through the same typed client that ``airflowctl`` uses. Replace any use of
``airflow.api.client`` with ``airflow.cli.api_client.get_cli_api_client``.
